基礎原子ってなに？

全ての物質は原子からできている。

原子の真ん中に原子核（陽子＋中性子）があって、そのまわりを電子が回っている。

3つだけ覚えろ

陽子：＋の電気。数＝原子番号。

中性子：電気なし。陽子と一緒に核にいる。

電子：−の電気。陽子と同じ数（中性原子のとき）。

質量数 A ＝ 陽子の数 Z ＋ 中性子の数
→ 重さの目安。電子は軽すぎて無視。
原子番号 ＝ 陽子の数 ＝ 電子の数
→ 原子番号だけで陽子も電子もわかる！

中性子の数 ＝ 質量数 − 原子番号 で求まる。

同位体（アイソトープ）

陽子数が同じで中性子数が違う原子のこと。

例：¹²C と ¹³C → 陽子は両方6個、中性子が6個と7個。

化学的性質はほぼ同じ！（電子の数が同じだから）

必須電子配置

電子は内側の殻から順番に入る。

K殻: 最大2個 → L殻: 最大8個 → M殻: 最大18個
各殻の最大 = 2n² （K:n=1, L:n=2, M:n=3）

最外殻の電子 = 価電子。これが化学的性質を決める。

Na(11): K(2) L(8) M(1) → 価電子1個 → 1個捨てたい → Na⁺になりやすい

Cl(17): K(2) L(8) M(7) → 価電子7個 → 1個欲しい → Cl⁻になりやすい

イオンのルール

陽イオン：電子を失う（Na → Na⁺、電子10個、Neと同じ配置で安定）

陰イオン：電子をもらう（Cl → Cl⁻、電子18個、Arと同じ配置で安定）

→ どっちも希ガスと同じ配置になって安定する！

核心3つの化学結合

イオン結合

金属が電子を渡し、非金属が受け取る。＋と−が引き合う。

例：NaCl, CaCl₂, MgO

性質：硬いがもろい。水に溶ける。融点高い。

共有結合

非金属どうしが電子を共有する。

分子を作る → 分子結晶（H₂O, CO₂）：柔らかい、融点低い

巨大な網目 → 共有結合の結晶（ダイヤ, SiO₂）：超硬い、融点超高い

金属結合

金属原子が自由電子を共有して結びつく。

例：Fe, Cu, Al

性質：電気・熱を通す。展性（たたくと広がる）・延性（引っ張ると伸びる）あり。

判別3ステップ

① 金属＋非金属 → イオン結合

② 非金属＋非金属 → 共有結合（分子 or 網目）

③ 金属だけ → 金属結合

電気陰性度：F > O > N > Cl > Br > C > S > H
差が大きい→イオン結合寄り、差が小さい→共有結合寄り

応用鉄の製錬（溶鉱炉）

自然界の鉄 = 酸化鉄 Fe₂O₃（赤錆みたいなもの）。これを還元して純粋な鉄にする。

① C + O₂ → CO₂
コークス（炭素）を燃やす
② CO₂ + C → 2CO
一酸化炭素ができる（これが主役！）
③ Fe₂O₃ + 3CO → 2Fe + 3CO₂
COがFe₂O₃からO(酸素)を奪う → 鉄が取り出せる！

銑鉄 → 鋼

銑鉄：溶鉱炉から出る鉄。炭素を約4%含む。硬いけどもろい。

鋼：転炉で酸素を吹き込み炭素を0.02〜2%に減らしたもの。丈夫でしなやか。

最重要mol（モル）

mol = 原子や分子を「まとめて数える単位」。1ダース=12個と同じノリ。

1 mol = 6.02 × 10²³ 個
アボガドロ定数。どんな物質でも1molはこの個数！

mol計算 3パターン

g → mol：÷ モル質量

mol → g：× モル質量

L → mol：÷ 22.4（標準状態の気体のみ）

モル質量 = 原子量の合計 (g/mol)
H₂O = 1×2+16 = 18 g/mol、CO₂ = 12+16×2 = 44 g/mol
気体 1 mol = 22.4 L（標準状態：0℃, 1気圧）
気体の種類に関係なく！ O₂でもCO₂でも1mol=22.4L

計算濃度

質量%濃度 = 溶質(g) ÷ 溶液(g) × 100
溶液 = 溶質 + 溶媒（全体で割る！）
モル濃度(mol/L) = mol ÷ 体積(L)
mLは÷1000してLに！ 500mL = 0.5L
密度→モル濃度: 10dw ÷ M
d=密度, w=質量%, M=モル質量

計算化学反応式の量的関係

係数の比 = molの比
これが全て！

例：2H₂ + O₂ → 2H₂O

→ H₂ : O₂ : H₂O = 2 : 1 : 2

→ H₂ が4molなら → O₂は2mol必要 → H₂Oは4molできる

手順3ステップ

① 問題の量をmolに変換（g÷モル質量、L÷22.4）

② 係数の比で目的物質のmolを求める

③ molをg or Lに変換して答え

核心酸化と還元

酸化 = 電子を失う = 酸化数が増える
還元 = 電子を得る = 酸化数が減る
酸化と還元は必ずセットで起きる！

酸化剤：相手を酸化する（自分は還元される）

還元剤：相手を還元する（自分は酸化される）

酸化数 5つのルール

① 単体 = 0（H₂, O₂, Fe）

② 化合物中の H = +1

③ 化合物中の O = −2

④ 化合物全体の合計 = 0

⑤ イオンの合計 = 電荷

計算方法：求めたい原子をxとして方程式を立てるだけ。

例：H₂SO₄ の S → 2(+1) + x + 4(−2) = 0 → x = +6

核心電池

電池 = 化学エネルギー → 電気エネルギー

負極で酸化（電子を出す）、正極で還元（電子を受ける）
覚え方：「ぼ（負）さん（酸化）」

イオン化傾向が大きい方が負極になる！

貸そうかな まぁ当てにすんな ひどすぎる借金

ダニエル電池

負極(Zn板)：Zn → Zn²⁺ + 2e⁻（亜鉛が溶ける）

正極(Cu板)：Cu²⁺ + 2e⁻ → Cu（銅が析出）

→ Znの方がイオン化傾向大きいからZnが負極

核心電気分解

電気分解 = 電気エネルギー → 化学変化（電池の逆）

陽極で酸化、陰極で還元
覚え方：「よう（陽）さん（酸化）」→ 電池と逆の極！

CuCl₂水溶液の電気分解

陰極：Cu²⁺ + 2e⁻ → Cu（銅が析出）

陽極：2Cl⁻ → Cl₂ + 2e⁻（塩素ガス発生）

ファラデーの法則：Q = It
Q=電気量(C), I=電流(A), t=時間(秒)
電子 1mol = 96500 C

気体の状態方程式

R = 8.3×10³ Pa·L/(mol·K)

(1) 27℃、1.0×10⁵ Pa で 8.3L の気体は何molか。

(2) 酸素 O₂ 6.4g を 27℃ で 5.0L の容器に入れたときの圧力を求めよ。(O=16)

27℃ → 300K（+273！）
(1) n = PV/(RT)
= (1.0×10⁵ × 8.3)/(8.3×10³ × 300)
= 830000/2490000
= 1/3 ≈ 0.33 mol
(2) O₂=32g/mol → 6.4÷32=0.2mol
P = nRT/V
= (0.2×8.3×10³×300)/5.0
= 498000/5
= 9.96×10⁴ ≈ 1.0×10⁵ Pa
PV=nRT → 求めたいものを左辺に！温度はK！
