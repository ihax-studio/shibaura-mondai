数学I データの分析

芝浦工大入試で出る！ データの分析はセンター・共通テストでほぼ毎年出題。平均値、分散、標準偏差、相関係数を確実に押さえよう。

必須公式まとめ（これだけ覚えろ！）

平均値 x̄ = (x₁ + x₂ + ... + xₙ) / n
→ 全部足して、個数で割るだけ！
分散 s² = (1/n)Σ(xᵢ - x̄)² = x²の平均 - (平均)²
→「各データと平均の差」を2乗して平均する。または「2乗の平均 ひく 平均の2乗」のほうが計算ラク！
標準偏差 s = √(分散)
→ 分散のルートを取るだけ！データのバラつき具合を表す。
共分散 sxy = (1/n)Σ(xᵢ - x̄)(yᵢ - ȳ) = xyの平均 - x̄ · ȳ
→ xとyの関係の強さ。プラスなら正の相関、マイナスなら負の相関。
相関係数 r = sxy / (sx · sy)　（−1 ≤ r ≤ 1）
→ 1に近い＝強い正の相関、−1に近い＝強い負の相関、0に近い＝相関なし
データ変換：u = ax + b のとき → ū = ax̄ + b、su = |a|·sx
→ 平均は a倍して b を足す。標準偏差は |a| 倍するだけ（bは関係なし）

問題 95 度数分布表

次のデータは、ある クラス40人の数学のテストの得点である。

階級(点)0~2020~4040~6060~8080~100
度数(人)3815104

(1) 最頻値（モード）を求めよ。

(2) 度数分布表から平均値を求めよ。

(3) 中央値（メジアン）が含まれる階級を答えよ。

解説（めっちゃ丁寧に！）

（1）最頻値（モード）
最頻値 = 度数が一番多い階級の「真ん中の値」
度数が一番多いのは「40~60」の15人
→ 階級値 = (40+60)/2 = 50点
（2）平均値
各階級の「階級値 × 度数」を全部足して、合計人数で割る

階級値：10, 30, 50, 70, 90
10×3 + 30×8 + 50×15 + 70×10 + 90×4
= 30 + 240 + 750 + 700 + 360 = 2080
平均値 = 2080 ÷ 40 = 52点
（3）中央値が含まれる階級
40人なので中央値は20番目と21番目の平均
度数を下から足していく：
0~20: 3人（累計3人）
20~40: 8人（累計11人）
40~60: 15人（累計26人） ← 20番目、21番目はここ！
→ 中央値が含まれる階級は 「40~60」
答え：(1) 50点　(2) 52点　(3) 40以上60未満の階級

問題 96 相関係数・散布図

次の表は、5人の生徒の数学と英語の得点である。

生徒ABCDE
数学 x5060708090
英語 y4055607570

(1) x と y の平均値をそれぞれ求めよ。

(2) x と y の共分散を求めよ。

(3) x と y の相関係数を求めよ。

解説

（1）平均値
x̄ = (50+60+70+80+90)/5 = 350/5 = 70
ȳ = (40+55+60+75+70)/5 = 300/5 = 60
（2）共分散
各(xᵢ-x̄)(yᵢ-ȳ)を計算：
A: (50-70)(40-60) = (-20)(-20) = 400
B: (60-70)(55-60) = (-10)(-5) = 50
C: (70-70)(60-60) = 0×0 = 0
D: (80-70)(75-60) = 10×15 = 150
E: (90-70)(70-60) = 20×10 = 200
共分散 = (400+50+0+150+200)/5 = 800/5 = 160
（3）相関係数
まずsx（xの標準偏差）を求める：
各(xᵢ-x̄)² : 400, 100, 0, 100, 400 → 分散 = 1000/5 = 200
sx = √200 = 10√2

sy（yの標準偏差）を求める：
各(yᵢ-ȳ)² : 400, 25, 0, 225, 100 → 分散 = 750/5 = 150
sy = √150 = 5√6

r = 160 / (10√2 × 5√6) = 160 / (50√12) = 160 / (100√3)
= 8/(5√3) = 8√3/15 ≈ 0.924
答え：(1) x̄=70, ȳ=60　(2) 160　(3) 8√3/15 ≈ 0.92（強い正の相関）

問題 97 分散・標準偏差

次のデータの分散と標準偏差を求めよ。

2, 4, 6, 8, 10

解説

Step1: 平均を出す
x̄ = (2+4+6+8+10)/5 = 30/5 = 6
Step2: 「2乗の平均」を出す
各データの2乗: 4, 16, 36, 64, 100
2乗の平均 = (4+16+36+64+100)/5 = 220/5 = 44
Step3: 分散 = 2乗の平均 − (平均)²
分散 = 44 − 6² = 44 − 36 = 8
Step4: 標準偏差 = √分散
標準偏差 = √8 = 2√2 ≈ 2.83
答え：分散 = 8、標準偏差 = 2√2

問題 98 データの変換

変量 x の平均が 50、標準偏差が 10 であるとき、u = 2x + 3 としたときの u の平均と標準偏差を求めよ。

解説

覚えるルール
u = ax + b のとき
・ū = a × x̄ + b（平均は a倍して b を足す）
・su = |a| × sx（標準偏差は |a| 倍、b は関係ない！）
計算
ū = 2 × 50 + 3 = 103
su = |2| × 10 = 20

超重要ポイント

標準偏差にはbを足さない！ 標準偏差は「バラつき」だから、全部に同じ数を足してもバラつきは変わらない。a倍だけが影響する！
答え：平均 = 103、標準偏差 = 20

化学基礎 原子の構造・電子配置・周期表

原子の中身を理解するのが化学の第一歩！ 陽子・中性子・電子の関係と、電子配置のルールを覚えよう。

必須知識

質量数 = 陽子の数 + 中性子の数
→ 質量数は原子の「重さ」の目安。陽子と中性子の合計。
原子番号 = 陽子の数 = 電子の数（中性原子のとき）
→ 原子番号がわかれば陽子と電子の数がわかる！
電子配置：K殻(2個) → L殻(8個) → M殻(18個) → N殻(32個)
→ 内側の殻から順番に埋めていく。各殻の最大収容数は 2n² 個。
同位体（アイソトープ）：陽子数が同じで中性子数が違う原子
→ 例：¹²C と ¹³C は同位体。化学的性質はほぼ同じ！
イオン：陽イオン = 電子を失う、陰イオン = 電子を受け取る
→ Na → Na⁺（電子1個失う）、Cl → Cl⁻（電子1個もらう）

化学基礎 化学結合

3つの結合（イオン・共有・金属）の違いを完璧に理解しよう！

3つの化学結合

イオン結合：金属 + 非金属 → 電子を渡して＋と−で引き合う
→ 例：NaCl（食塩）。Naが電子をClに渡す。
共有結合：非金属 + 非金属 → 電子を共有する
→ 例：H₂O, CO₂。電子を一緒に使う。
金属結合：金属同士 → 自由電子が金属原子を結びつける
→ 例：Fe, Cu。電気を通す理由は自由電子！
電気陰性度：F > O > N > Cl > Br > C > S > H
→ 電子を引っ張る力の強さ。Fが最強！差が大きいとイオン結合になる。

問題 259 化学結合の種類

次の物質の化学結合の種類を答えよ。また、それぞれの結晶の名称を答えよ。

(1) NaCl　(2) H₂O　(3) Fe　(4) SiO₂　(5) CO₂　(6) Cu　(7) CaCl₂　(8) ダイヤモンド C

解説

見分け方のコツ
① 金属 + 非金属 → イオン結合 → イオン結晶
② 非金属 + 非金属 → 共有結合
　→ 分子を作る → 分子結晶（CO₂, H₂Oなど）
　→ 巨大な網目 → 共有結合の結晶（ダイヤ, SiO₂）
③ 金属のみ → 金属結合 → 金属結晶
(1) NaCl → Na(金属) + Cl(非金属) → イオン結合・イオン結晶
(2) H₂O → H(非金属) + O(非金属) → 共有結合・分子結晶
(3) Fe → 金属のみ → 金属結合・金属結晶
(4) SiO₂ → 非金属同士だが巨大網目構造 → 共有結合の結晶
(5) CO₂ → 非金属同士、分子を作る → 共有結合・分子結晶
(6) Cu → 金属のみ → 金属結合・金属結晶
(7) CaCl₂ → Ca(金属) + Cl(非金属) → イオン結合・イオン結晶
(8) ダイヤモンド C → 炭素原子の巨大網目 → 共有結合の結晶

結晶の性質まとめ

・イオン結晶：硬いがもろい、水に溶ける、融点高い
・分子結晶：柔らかい、融点低い（ドライアイスなど）
・共有結合の結晶：超硬い、融点超高い（ダイヤモンド）
・金属結晶：電気通す、展性・延性あり

化学基礎 物質量（mol）と化学反応式

molは化学の計算の基本！ ここを落とすと全部ダメになるから、徹底的に練習しよう。

mol計算の超重要公式

1 mol = 6.02 × 10²³ 個（アボガドロ定数）
→ 原子でも分子でも、1molはこの個数！
mol = 質量(g) ÷ モル質量(g/mol)
→ 例：水H₂O のモル質量 = 1×2+16 = 18 g/mol。36gの水 = 36÷18 = 2mol
気体：1 mol = 22.4 L（標準状態 0℃, 1atm）
→ 気体の種類に関係なく、1molは22.4L！
mol = 体積(L) ÷ 22.4
モル濃度(mol/L) = mol ÷ 溶液の体積(L)
質量パーセント濃度(%) = (溶質の質量 / 溶液の質量) × 100

問題 121 mol計算の基本

次の各問いに答えよ。原子量：H=1, C=12, N=14, O=16, S=32, Na=23, Cl=35.5

(1) 水 H₂O 90g は何molか。

(2) 二酸化炭素 CO₂ 2mol の質量は何gか。

(3) 標準状態で 5.6L の酸素O₂は何molか。

(4) 硫酸 H₂SO₄ 0.5mol 中の酸素原子は何個か。

(5) 0.2mol/L の NaOH 水溶液 500mL 中の NaOH は何molか。

解説

(1) H₂O 90g → mol
H₂Oのモル質量 = 1×2 + 16 = 18 g/mol
mol = 90 ÷ 18 = 5 mol
(2) CO₂ 2mol → g
CO₂のモル質量 = 12 + 16×2 = 44 g/mol
質量 = 2 × 44 = 88 g
(3) 5.6L のO₂ → mol
mol = 5.6 ÷ 22.4 = 0.25 mol
(4) H₂SO₄ 0.5mol中の酸素原子の個数
H₂SO₄ 1分子中に O は 4個ある
0.5mol の H₂SO₄ に含まれる O = 0.5 × 4 = 2 mol
個数 = 2 × 6.02×10²³ = 1.204 × 10²⁴ 個
(5) 0.2mol/L × 500mL
500mL = 0.5L
mol = 0.2 × 0.5 = 0.1 mol

mol計算3ステップ

① まずモル質量を計算する（原子量の合計）
② 単位に注目（g→mol、mol→g、L→mol）
③ mLはLに直す（÷1000）のを忘れない！
答え：(1) 5mol (2) 88g (3) 0.25mol (4) 1.204×10²⁴個 (5) 0.1mol

問題 122 化学反応式の量的関係

次の化学反応について答えよ。

2H₂ + O₂ → 2H₂O

(1) 水素 4mol を完全に反応させるのに必要な酸素は何molか。

(2) 水素 8g を完全燃焼させると、水は何g生成するか。

(3) 標準状態で酸素 11.2L と反応する水素は何Lか。

解説

化学反応式の係数 = molの比
2H₂ + O₂ → 2H₂O
H₂ : O₂ : H₂O = 2 : 1 : 2
(1)
H₂ : O₂ = 2 : 1
H₂ 4mol のとき O₂ = 4 × (1/2) = 2 mol
(2)
H₂ 8g → mol: 8 ÷ 2 = 4mol
H₂ : H₂O = 2 : 2 = 1 : 1 なので H₂O も 4mol
H₂O 4mol → g: 4 × 18 = 72 g
(3)
O₂ 11.2L → mol: 11.2 ÷ 22.4 = 0.5mol
H₂ : O₂ = 2 : 1 → H₂ = 0.5 × 2 = 1mol
H₂ 1mol → L: 1 × 22.4 = 22.4 L
答え：(1) 2mol (2) 72g (3) 22.4L

問題 171 溶液の濃度

次の問いに答えよ。

(1) 食塩 20g を水 180g に溶かした。質量パーセント濃度を求めよ。

(2) 密度 1.2 g/cm³ の 10% 塩酸のモル濃度を求めよ。（HCl=36.5）

解説

(1)
質量パーセント濃度 = 溶質/溶液 × 100
溶液 = 溶質 + 溶媒 = 20 + 180 = 200g
濃度 = 20/200 × 100 = 10%
(2) 密度→モル濃度の変換（超頻出！）
溶液1Lを考える（= 1000 cm³）
質量 = 1000 × 1.2 = 1200g
溶質(HCl)の質量 = 1200 × 0.10 = 120g
mol = 120 ÷ 36.5 ≈ 3.29 mol
→ モル濃度 ≈ 3.29 mol/L

密度→モル濃度の公式

モル濃度 = (1000 × 密度 × 質量パーセント/100) ÷ モル質量
= 1000dw / (100M) = 10dw / M
答え：(1) 10% (2) 約3.29 mol/L

化学基礎 酸化還元・電池・電気分解

酸化還元は電子の動きに注目！「酸化数」を使いこなせば怖くない。

酸化還元の超重要ルール

酸化 = 電子を失う = 酸化数が増える
還元 = 電子を得る = 酸化数が減る
→ 「酸化されたやつが還元剤」「還元されたやつが酸化剤」（逆に覚えないで！）
酸化数のルール：
① 単体 = 0　② H = +1（金属水素化物は−1）
③ O = −2（過酸化物は−1）　④ 化合物全体 = 0　⑤ イオン = 電荷
電池：負極で酸化、正極で還元
→ 負極から電子が出ていく。イオン化傾向が大きい方が負極。
電気分解：陽極で酸化、陰極で還元
→ 電池の逆！外部から電気を流して無理やり反応させる。
ファラデーの法則：電気量 Q = It（C）、1mol電子 = 96500C

問題 260 酸化数の決定

次の下線部の原子の酸化数を求めよ。

(1) Na₂O　(2) NH₃　(3) H₂SO₄　(4) KMnO₄　(5) Cr₂O₇²⁻

解説

解き方
「化合物全体の酸化数の合計 = 0（イオンなら電荷）」を方程式にする！
(1) Na₂O
Na₂O: Na(x)×2 + O(−2) = 0
2x − 2 = 0 → x = +1
(2) NH₃
N(x) + H(+1)×3 = 0
x + 3 = 0 → x = −3
(3) H₂SO₄
H(+1)×2 + S(x) + O(−2)×4 = 0
2 + x − 8 = 0 → x = +6
(4) KMnO₄
K(+1) + Mn(x) + O(−2)×4 = 0
1 + x − 8 = 0 → x = +7
(5) Cr₂O₇²⁻
Cr(x)×2 + O(−2)×7 = −2（イオンだから電荷 = −2）
2x − 14 = −2 → 2x = 12 → x = +6
答え：(1) +1 (2) −3 (3) +6 (4) +7 (5) +6

問題 261 電池と電気分解

(1) ダニエル電池について、負極・正極の反応をそれぞれ書け。素材：Zn板とCu板、ZnSO₄水溶液とCuSO₄水溶液

(2) 塩化銅(II) CuCl₂水溶液を炭素電極で電気分解した。陽極と陰極で起こる反応を書け。

(3) 0.50A の電流を 965秒間流したとき、陰極に析出する銅は何gか。（Cu=64）

解説

(1) ダニエル電池
イオン化傾向：Zn > Cu なので Zn が負極
負極（Zn板）：Zn → Zn²⁺ + 2e⁻（酸化）
正極（Cu板）：Cu²⁺ + 2e⁻ → Cu（還元）
→ Znが溶けて、Cuが析出する！
(2) CuCl₂水溶液の電気分解
陰極：Cu²⁺ + 2e⁻ → Cu（銅が析出）
陽極：2Cl⁻ → Cl₂ + 2e⁻（塩素ガスが発生）
(3) ファラデーの法則で計算
Q = It = 0.50 × 965 = 482.5 C
電子のmol = 482.5 ÷ 96500 = 0.005 mol
Cu²⁺ + 2e⁻ → Cu より、Cu = 0.005 ÷ 2 = 0.0025 mol
質量 = 0.0025 × 64 = 0.16 g

電池 vs 電気分解の覚え方

・電池は「ぼ（負）さん（酸化）」→ 負極で酸化
・電気分解は「よう（陽）さん（酸化）」→ 陽極で酸化
→ 電池と電気分解では酸化する極が逆！
答え：(1) 上記参照 (2) 上記参照 (3) 0.16g

化学 結晶構造・気体の法則

結晶構造の計算と気体の状態方程式は芝浦工大でよく出る！

結晶構造の公式

体心立方格子：含まれる原子数 = 2個
配位数 = 8、充填率 = 68%
面心立方格子：含まれる原子数 = 4個
配位数 = 12、充填率 = 74%
六方最密構造：含まれる原子数 = 2個
配位数 = 12、充填率 = 74%
数え方のコツ：
・頂点の原子 → 1/8個分（8つの格子で共有）
・辺の原子 → 1/4個分
・面の原子 → 1/2個分
・体内の原子 → 1個分

気体の法則

理想気体の状態方程式：PV = nRT
P=圧力(Pa), V=体積(L), n=mol, R=8.31(気体定数), T=絶対温度(K)
T(K) = t(℃) + 273
ボイルの法則：P₁V₁ = P₂V₂（温度一定）
シャルルの法則：V₁/T₁ = V₂/T₂（圧力一定）
ボイル・シャルルの法則：P₁V₁/T₁ = P₂V₂/T₂

問題 202, 203 結晶構造

(1) 体心立方格子の単位格子に含まれる原子の数を求めよ。

(2) 面心立方格子の単位格子に含まれる原子の数を求めよ。

(3) 面心立方格子の一辺の長さが a cm、原子量が M、アボガドロ定数が Nₐ のとき、結晶の密度を求めよ。

解説

(1) 体心立方格子
頂点：8個 × 1/8 = 1個
体心：1個 × 1 = 1個
合計 = 2個
(2) 面心立方格子
頂点：8個 × 1/8 = 1個
面心：6個 × 1/2 = 3個
合計 = 4個
(3) 密度の計算
単位格子の体積 = a³ cm³
含まれる原子 = 4個 = 4/Nₐ mol
質量 = (4/Nₐ) × M = 4M/Nₐ (g)
密度 = 質量/体積 = 4M / (Nₐ × a³) g/cm³
答え：(1) 2個 (2) 4個 (3) 4M/(Nₐa³) g/cm³

問題 210 気体の状態方程式

次の問いに答えよ。R = 8.3×10³ Pa·L/(mol·K)

(1) 27℃、1.0×10⁵ Pa で 8.3L の気体は何molか。

(2) 酸素O₂ 6.4g を 27℃で 5.0L の容器に入れたときの圧力を求めよ。（O=16）

解説

まず温度をケルビンに変換！
27℃ → 27 + 273 = 300 K
(1) PV = nRT → n = PV/(RT)
n = (1.0×10⁵ × 8.3) / (8.3×10³ × 300)
= 8.3×10⁵ / (2.49×10⁶)
= 1/3 ≈ 0.33 mol
(2) まずmolを求める
O₂のモル質量 = 32 g/mol
n = 6.4 ÷ 32 = 0.2 mol

PV = nRT → P = nRT/V
P = (0.2 × 8.3×10³ × 300) / 5.0
= (0.2 × 2490000) / 5.0
= 498000 / 5.0
= 9.96×10⁴ ≈ 1.0×10⁵ Pa

PV=nRT 使い方3ステップ

① 温度は必ず K（ケルビン）に直す！ +273を忘れるな
② 求めたいものを左辺に持ってくる
③ 単位を揃える（Pa, L, mol, K）
答え：(1) 1/3 mol (2) 約1.0×10⁵ Pa
